USA Airport Operations & Delay Analysis Dashboard
          // Project Overview

This project is a multi-page Power BI dashboard analysing over 36,000 U.S. flight records to uncover operational performance insights related to:
Flight cancellations
Departure & arrival delays
Distance vs delay behaviour
Monthly & weekday patterns
Origin & destination performance
The goal of this project was to transform raw flight data into clear, interactive, and insight-driven dashboards suitable for operational and executive decision-making.

          Tools & Technologies Used
Power BI (Data Modelling & Visualisation)
DAX (Measures & KPIs)
Power Query (Data Cleaning & Transformation)
Excel (Initial Data Handling)
SQL Concepts (Data structuring logic)

        // Data Cleaning & Preparation
Key data preparation steps included:
Converting date/time columns to the proper datetime format
Creating calculated delay fields (Departure & Arrival Delay in minutes)
Handling missing Origin values (categorised as “Unknown” to preserve data integrity)
Creating custom columns for:
Month Name
Day Name
Year
Building DAX measures for KPIs such as:
Total Flights
Cancelled Flights
Completed Flights
Average Departure Delay
Average Arrival Delay
% Flights Delayed

       // Dashboard Structure
       Executive Overview Page
Designed for high-level performance review:
Total Flights KPI
Cancelled Flights KPI
Monthly Flight Trends
Cancellations by Weekday
Interactive filters (Month, Year, Origin, Destination)

       / Delay & Time Analysis Page
Focused on delay behaviour analysis:
Avg Departure Delay by Distance (Scatter Plot with Trend Line)
Avg Arrival Delay by Distance
Geographic Map of Origin & Destination Performance
Delayed Flights Count
% Flights Delayed KPI
Insight: No strong linear correlation was observed between flight distance and delay magnitude.
     
      /Detailed Flight Analysis Page
Designed for deeper operational inspection:
Detailed flight-level table
Monthly Delay Trend Line (Arrival vs Departure)
Top 10 Destinations by Cancellations
Drill-through analysis capability

          // Key Insights
36K+ flights analyzed
453 total cancellations
24% of flights experienced measurable delay
Certain destinations show significantly higher cancellation frequency
Delay patterns fluctuate seasonally
Distance alone does not strongly predict delay

        // Business Value
This dashboard demonstrates how operational data can be transformed into actionable insights for:
Airline performance monitoring
Airport operational planning
Route risk assessment
Delay reduction strategies
Executive reporting
